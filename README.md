# Felipe Mammana Portfolio

Portfolio pessoal premium em Next.js, React, TypeScript, Tailwind CSS, Framer Motion e Lucide Icons.

## Conceito

O site apresenta Felipe Mammana como desenvolvedor backend-focused/fullstack em desenvolvimento, com foco em sistemas reais, automacoes, projetos internos, interfaces modernas e experimentos com IA.

## Rodando localmente

```bash
npm install
npm run dev
```

Depois acesse `http://localhost:3000`.

## Estrutura

- `src/app/page.tsx`: pagina single page e componentes principais.
- `src/app/globals.css`: design system, paleta, animacoes e responsividade.
- `src/data/projects.ts`: dados dos projetos.
